# Spandex

A jQuery plugin for stretchy backgrounds.

__current status:__ in development

Tested:

* Chrome
* Fire fox
* ie7
* Safari
* iOS


## Basic usage
```javascript
$("body").spandex({use: "/images/studio54.jpg"});
```

See `tests` for other examples.
