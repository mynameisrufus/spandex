# Spandex

A jQuery and Zepto plugin for stretchy backgrounds.

__current status:__ in development

Tested:

* Chrome
* Fire fox
* ie7
* Safari
* iOS

## Basic usage
```javascript
$("body").spandex("/images/studio54.jpg");
```

See `tests` for other examples.
